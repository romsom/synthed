Class: ParamByte parameter.py

Sysex parameter values are packed into bytes, possibly across byte boundaries. A Param references one or more ParamBytes. Each ParamByte holds a contiguous range of bits in a byte. This corresponds to the <byte> entity in "patch.dtd".
